# SmartTask Management System

A modern full-stack task management application built using ASP.NET Core Web API and React.

## Features

- User Registration & Login
- JWT Authentication
- Project Management
- Task Management
- Task Priority & Status Tracking
- Secure REST API
- SQL Server Database
- Responsive UI

## Technologies Used

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

### Frontend
- React
- Vite
- Axios
- Bootstrap
- React Router DOM
